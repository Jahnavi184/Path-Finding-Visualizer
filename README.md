# Path-Finding-Visualizer

An interactive web application built using **React** and **Vite** that animates classic graph algorithms on a dynamic grid. The tool helps users intuitively understand how different search algorithms explore a graph and calculate the shortest path in real time.

## 🚀 Features
* **Algorithm Traversal:** Observe how **Dijkstra's Algorithm**, **Breadth-First Search (BFS)**, and **Depth-First Search (DFS)** explore nodes.
* **Interactive Grid:** Dynamically create custom walls/obstacles, move the start and end nodes, and watch the grid recalculate.
* **Real-Time Animations:** Clear, visual indicators tracking visited nodes and the final optimal shortest path.
* **Optimized Rendering:** Built using React functional components, modern hooks, and a clean, modular JavaScript architecture.

## 🛠️ Tech Stack
* **Frontend:** React, JavaScript (ES6+), HTML5, CSS3
* **Build Tool:** Vite
